# Dream Book Shop Data Analyzer

A Python 3 CLI application that analyzes the "Dataset Books.csv" file and produces summaries with optional charts.

## Features
- Publication trends over time
- Top 5 most prolific authors
- Language distribution (counts + percentages)
- Publisher counts
- Missing ISBN analysis
- Year-by-language breakdown

## Setup

```bash
pip install -r requirements.txt
```

Place `Dataset Books.csv` in the project root (same directory as `app.py`).

## Run

```bash
python app.py
```

Charts are saved in the `output/` directory.
